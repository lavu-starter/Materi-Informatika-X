## ðŸ”Ž Apa itu Percabangan?

Percabangan adalah **logika pengambilan keputusan** dalam program.
Ibarat jalan bercabang: kalau kondisi tertentu benar â†’ ambil jalan A, kalau tidak â†’ ambil jalan B.

Contoh sehari-hari:

* Kalau lapar â†’ makan.
* Kalau hujan â†’ bawa payung, kalau tidak â†’ cukup pakai topi.

Contoh:

## ðŸ”„ Percabangan `if-else`

Jika kondisi **benar (True)** â†’ jalankan blok `if`.
Jika kondisi **salah (False)** â†’ jalankan blok `else`.

```python

    print("Boleh membuat KTP.")
else:
    print("Belum boleh membuat KTP.")
```

## ðŸ”€ Percabangan `if-elif-else`

Untuk banyak kondisi, gunakan `elif` (else if).

```python

    print("Nilai kamu A")

    print("Nilai kamu B")

    print("Nilai kamu C")
else:
    print("Nilai kamu D")
```
