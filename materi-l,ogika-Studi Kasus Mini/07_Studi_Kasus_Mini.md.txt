---

## 🔎 Studi Kasus: Program Penilaian Siswa

Program ini akan:

1. Meminta **nama** dan **jumlah mata pelajaran**.
2. Meminta **nilai setiap mata pelajaran**.
3. Menghitung **rata-rata nilai**.
4. Menentukan **kategori nilai**:

   * ≥ 90 → A
   * 75–89 → B
   * 60–74 → C
   * <60 → D
5. Menampilkan **laporan akhir**.

---

## 🖊️ Algoritma Singkat

1. Input nama siswa.
2. Input jumlah mata pelajaran.
3. Buat list kosong untuk menampung nilai.
4. Gunakan perulangan untuk memasukkan nilai setiap mata pelajaran.
5. Hitung rata-rata nilai.
6. Gunakan percabangan untuk menentukan kategori nilai.
7. Tampilkan laporan siswa lengkap.

---

## 🧠 Latihan Logika

1. Modifikasi program agar juga menampilkan **nilai tertinggi dan terendah**.
2. Tambahkan **fungsi** untuk menghitung rata-rata dan kategori nilai.
3. Tambahkan dictionary untuk menyimpan data siswa secara lengkap: nama, nilai, rata-rata, dan kategori.
