Daripada menulis kode berulang, kita bisa menggunakan loop.

## ğŸ–Šï¸ Perulangan `for`

Digunakan untuk mengulang sebanyak jumlah tertentu.

Contoh:

```python
for i in range(5):
    print("Halo Python")
```

Hasil:

```
Halo Python
Halo Python
Halo Python
Halo Python
Halo Python
```

### Perulangan dengan angka

```python
for i in range(1, 6):
    print("Angka:", i)
```

Hasil:

```
Angka: 1
Angka: 2
Angka: 3
Angka: 4
Angka: 5
```

## ğŸ”„ Perulangan `while`

Digunakan selama kondisi masih **True**.

Contoh:

```python
angka = 1
while angka <= 5:
    print("Angka:", angka)
    angka += 1
```

Hasil:

```
Angka: 1
Angka: 2
Angka: 3
Angka: 4
Angka: 5
```
