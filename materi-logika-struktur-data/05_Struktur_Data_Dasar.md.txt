---

## 🔎 Mengapa Perlu Struktur Data?

Kalau hanya 1 nilai, cukup dengan variabel biasa:

```python
nilai1 = 80
nilai2 = 85
nilai3 = 90
```

Tapi kalau banyak data, akan **repot**.
Solusinya → gunakan **struktur data**.

---

## 🗂️ List

List digunakan untuk menyimpan **banyak data** dan bisa diubah (mutable).
Ditulis dengan **\[ ]**.

Contoh:

```python
buah = ["apel", "jeruk", "mangga"]
print(buah[0])   # apel
buah.append("pisang")  # tambah data
print(buah)
```

---

## 🗂️ Tuple

Tuple mirip List, tapi **tidak bisa diubah** (immutable).
Ditulis dengan **( )**.

Contoh:

```python
warna = ("merah", "hijau", "biru")
print(warna[1])   # hijau
# warna[0] = "kuning" ❌ (tidak bisa diubah)
```

---

## 🗂️ Dictionary

Dictionary menyimpan data dalam bentuk **key : value**.
Ditulis dengan **{ }**.

Contoh:

```python
siswa = {
    "nama": "Budi",
    "umur": 16,
    "kelas": "XI"
}
print(siswa["nama"])  # Budi
```

3. Buat dictionary `buku` dengan key: judul, penulis, tahun. Cetak semua datanya.

---

## 🏁 Kesimpulan

* **List** → banyak data, bisa diubah.
* **Tuple** → banyak data, tidak bisa diubah.
* **Dictionary** → data pasangan key : value.

Struktur data memudahkan kita menyimpan dan mengolah banyak informasi.
