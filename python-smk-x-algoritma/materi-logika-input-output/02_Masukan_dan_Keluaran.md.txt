## ğŸ–Šï¸ Masukan (Input)

Di Python, masukan dilakukan dengan fungsi `input()`.

Contoh:

```python
nama = input("Masukkan nama kamu: ")
print("Halo,", nama)
```

Jika kita ketik `Rina`, hasilnya:

```
Masukkan nama kamu: Rina
Halo, Rina
```

> âš¡ Catatan: Semua hasil dari `input()` **selalu berupa string** (teks), walaupun kita ketik angka.

## ğŸ“¤ Keluaran (Output)

Keluaran biasanya menggunakan fungsi `print()`.

Contoh:

```python
print("Belajar Python itu mudah!")
```

Bisa juga menampilkan beberapa nilai sekaligus:

```python
nama = "Budi"
umur = 16
print("Nama:", nama, "| Umur:", umur)
```

Hasilnya:

```
Nama: Budi | Umur: 16
```

---

## ğŸ”„ Mengolah Input

Karena `input()` menghasilkan **string**, kalau kita ingin angka, harus diubah (konversi).

Contoh:

```python
angka = int(input("Masukkan angka: "))
print("Angka dikali 2 =", angka * 2)
```

Jika kita ketik `5`, hasilnya:

```
Masukkan angka: 5
Angka dikali 2 = 10
```
