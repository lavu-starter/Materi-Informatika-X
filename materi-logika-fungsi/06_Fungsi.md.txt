## ğŸ” Apa itu Fungsi?

Fungsi adalah **sepotong kode yang bisa dipanggil ulang** untuk melakukan tugas tertentu.
Ibaratnya seperti **alat atau mesin**, kita tinggal memanggilnya tanpa menulis kode dari awal.

Contoh sehari-hari:

* Mesin kopi â†’ tinggal tekan tombol, kopi siap.
* Kalkulator â†’ tinggal tekan angka dan operasi, hasil keluar.

## ğŸ–Šï¸ Membuat Fungsi Sederhana

Struktur dasar:

```python
def nama_fungsi():
    # kode yang dijalankan
    print("Halo, dunia!")
```

Memanggil fungsi:

```python
nama_fungsi()  # akan menampilkan "Halo, dunia!"
```

---

## ğŸ”„ Fungsi dengan Parameter

Parameter adalah **data yang dikirim ke fungsi**.

Contoh:

```python
def sapa(nama):
    print("Halo,", nama)

sapa("Budi")  # Output: Halo, Budi
```

---

## â• Fungsi dengan Nilai Kembalian (Return)

Fungsi bisa mengembalikan hasil menggunakan `return`.

Contoh:

```python
def kali(a, b):
    return a * b

hasil = kali(5, 3)
print("Hasil perkalian:", hasil)  # Output: 15
```
